# Cheat Sheet: Python Data Structures Part-2

## Dictionaries

| Package/Method | Description | Code Example |
|----------------|-------------|--------------|
| **Creating a Dictionary** | A dictionary in Python is a built-in data structure that stores information in key-value pairs. Each key must be unique, and it is used to access its corresponding value. Dictionaries are enclosed in curly braces `{}`, and each pair is separated by a comma. | ```python<br>dict_name = {} # Creates an empty dictionary<br>person = {"name": "John", "age": 30, "city": "New York"}<br>``` |
| **Accessing Values** | You can access the values in a dictionary using their corresponding keys.<br><br>**Syntax:**<br>`Value = dict_name["key_name"]` | ```python<br>name = person["name"]<br>age = person["age"]<br>``` |
| **Add or modify** | You can add or modify elements in a dictionary by assigning a value to a key using the assignment operator `=`. When you assign a value to a key that does not already exist, Python automatically creates a new key-value pair and adds it to the dictionary. However, if the key already exists, the existing value will be replaced with the new one.<br><br>**Syntax:**<br>`dict_name[key] = value` | ```python<br>person["Country"] = "USA" # A new entry will be created.<br>person["city"] = "Chicago" # Update the existing value for the same key<br>``` |
| **del** | Removes the specified key-value pair from the dictionary. Raises a KeyError if the key does not exist.<br><br>**Syntax:**<br>`del dict_name[key]` | ```python<br>del person["Country"]<br>``` |
| **key existence** | You can check for the existence of a key in a dictionary using the `in` keyword | ```python<br>if "name" in person:<br>    print("Name exists in the dictionary.")<br>``` |
| **keys()** | Retrieves all keys from the dictionary and converts them into a list. Useful for iterating or processing keys using list methods.<br><br>**Syntax:**<br>`keys_list = list(dict_name.keys())` | ```python<br>person_keys = list(person.keys())<br>``` |
| **values()** | Extracts all values from the dictionary and converts them into a list. This list can be used for further processing or analysis.<br><br>**Syntax:**<br>`values_list = list(dict_name.values())` | ```python<br>person_values = list(person.values())<br>``` |
| **items()** | Retrieves all key-value pairs as tuples and converts them into a list of tuples. Each tuple consists of a key and its corresponding value.<br><br>**Syntax:**<br>`items_list = list(dict_name.items())` | ```python<br>info = list(person.items())<br>``` |

## Sets

| Package/Method | Description | Code Example |
|----------------|-------------|--------------|
| **Defining Sets** | A set in Python is an unordered collection that stores unique elements, meaning it automatically removes duplicates. Sets are enclosed in curly braces `{}` and are commonly used when you need to store a group of distinct values or perform mathematical operations such as union, intersection, and difference.<br><br>Because sets are unordered, the elements have no fixed position or index, and their order may change each time you print or iterate over them. | ```python<br>empty_set = set() # Creating an Empty Set<br>fruits = {"apple", "banana", "orange"}<br>colors = {"orange", "red", "green"}<br>```<br><br>**Note:** These two sets will be used in the examples that follow. |
| **add()** | Add elements to a set. You can use the `add()` method to insert a new element into a set. If the element already exists, it will not be added again, since sets automatically store only unique values.<br><br>**Syntax:**<br>`set_name.add(element)` | ```python<br>fruits.add("mango")<br>``` |
| **in** | Checks if an element exists in a set. Use the `in` keyword to verify whether a specific element is present in a set. It returns `True` if the element exists, and `False` otherwise.<br><br>**Syntax:**<br>`element in set_name` | ```python<br>"banana" in fruits<br>``` |
| **issubset()** | Check if one set is a subset of another. The `issubset()` method is used to determine whether all elements of one set exist within another set. If every element of the current set is found in the specified set, it returns `True`; otherwise, it returns `False`.<br><br>**Syntax:**<br>`is_subset = set1.issubset(set2)` | ```python<br>is_subset = fruits.issubset(colors)<br>``` |
| **issuperset()** | Check if one set is a superset of another. The `issuperset()` method is used to verify whether the current set contains all elements of another set. It returns `True` if every element of the specified set exists within the current set; otherwise, it returns `False`.<br><br>**Syntax:**<br>`is_superset = set1.issuperset(set2)` | ```python<br>is_superset = colors.issuperset(fruits)<br>``` |
| **remove()** | Remove a specific element from a set. The `remove()` method is used to delete a particular element from a set. If the element you try to remove does not exist, Python will raise a `KeyError`, which means you must ensure the element is present before calling this method.<br><br>This method is useful when you are certain that the item you want to remove is already in the set.<br><br>**Syntax:**<br>`set_name.remove(element)` | ```python<br>fruits.remove("banana")<br>``` |
| **Set Operations** | Perform various operations on sets: union, intersection, difference, symmetric difference.<br><br>**Syntax:**<br>`union_set = set1.union(set2)`<br>`intersection_set = set1.intersection(set2)`<br>`difference_set = set1.difference(set2)`<br>`sym_diff_set = set1.symmetric_difference(set2)` | ```python<br>combined = fruits.union(colors)<br>common = fruits.intersection(colors)<br>unique_to_fruits = fruits.difference(colors)<br>sym_diff = fruits.symmetric_difference(colors)<br>``` |
